The Journey begins ...

I started to get into touch with the coding world when I was in college. It always fascinated me how just some lines of code can create something so complex and useful. I started to learn Python from a 12 Hour Youtube tutorial Bro Code. From there, I learned the basic principles (classes, if/else statements ... etc), and created little programs such as a calculator or a timer or even little mini games such as snake game, tic tac toe or rock paper scissors ! After getting fundamental knowledge of Python, I got into Web Development. I started to learn HTML, CSS and JavaScript. My First Website contained lots of minigames which I played with friends during class. Web Development / Web Design is an area which interests me a lot since it is closely related to being creative and innovative.

First Big Project

My first big Project, probably the biggest Project I ever made, was creating my own Artificial Intelligence. In Switzerland, doing a project work was part of earning the Matura degree. For me, it was clear that I wanted to do something with coding. After exploring project ideas with my supervisor, we came up with following Project Title : "Designig chess playing programs using deep learning algorithms". Basically, I trained a model using a Convolutional Neural Network to play chess. I used a dataset of 2.5 million chess games and trained the model for 10 hours on Google Colab. Furthermore, I got into touch with modules like pandas, numpy, tensorflow, keras and chess. I used Pygame to visualize the chess board and play against the model.
