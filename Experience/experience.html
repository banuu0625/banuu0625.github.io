I started to get into touch with the coding world when I was in College. It always fascinated me how just some lines of code can create something so complex and useful. I started to learn Python from a 12 Hour Tutorial from Youtube from Bro Code. Like this, I created first minigames such as sneak or a calculator. After getting fundamental knowledge of Python, I got into Web Development. I started to learn HTML, CSS and JavaScript. My First Website contained lots of minigames which I played with friends during physics class. Unfortunately, I lost the code of this website, but I still have some Pictures.

First Big Project

My first big Project, probably the biggest Project I ever made, was creating my own Artificial Intelligence. In Switzerland, doing a project work was part of earning the Matura degree. For me, it was clear that I wanted to do something with coding. After exploring project ideas with my supervisor, we came up with following Project Title : "Designig chess playing programs using deep learning algorithms". Basically, I trained a model using a Convolutional Neural Network to play chess. I used a dataset of 2.5 million chess games and trained the model for 10 hours on Google Colab. Furthermore, I got into touch with modules like pandas, numpy, tensorflow, keras and chess. I used Pygame to visualize the chess board and play against the model.
